# Python Automation

A small collection of Python scripts created to automate repetitive file-related tasks.

This repository is focused on building simple utilities that save time during data preparation and document processing workflows.

## Automation built so far

### 1. Batch File Renamer

A script that automatically renames multiple files inside a folder using a sequential naming format.

### 2. Job Automatic Leave form and Mail Sender

Currently workng on it...

### 3. Stock History range (2012-01-01 to 2026 onwards) Automation

It include getting price history from [neps](https://nepsealpha.com/) and it also include script to get json file for the dead stocks from https://www.sharesansar.com/.

Additionaly it include the utility script to convert the json format into the csv format for easier data collection.
